# Cub3D
🎮

Graphic design project made with minilibX.
See [Wolfenstein 3D](http://users.atw.hu/wolf3d/).

## Run the project
* git clone the project
* make
* ./cub3D map.cub

## Commands
* W, A, S, D to move ;
* ESC to close the window and exit the program ;
* SPACE to jump ;
* O to crawl ;
* P to run ;
* → to turn right ;
* ← to turn left ;
* ↑	to look up ;
* ↓ to look down.
